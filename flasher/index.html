⚠️ Requirements

  • Hardware: ESP32-2432S028R (CYD) or compatible ESP32 with TFT display
  • Browser: Chrome, Edge, or Opera (with Web Serial API support)
  • Cable: USB-C data cable (not charge-only)
  • Driver: CH340/CP2102 USB driver (usually auto-installed)

🔥 Flash Firmware

🎨 Color Inversion Fix

Some CYD displays show inverted colors - if this happens to you, use this fix!

Do you see this?

  • ❌ Text appears BLACK on WHITE background
  • ❌ Colors look inverted/wrong
  • ❌ Display is hard to read
  • ❌ Interface doesn't look right

Instructions: Try the normal firmware first. If colors are inverted, come back and install this fix.

✅ After fix: Text should be WHITE on BLACK background with correct colors

💡 What Happens Next?

After flashing, your ESP32 will restart and create a WiFi network called "MakerWorld-Setup". Connect to it with your phone or computer, and a configuration page will open automatically. Select your home WiFi network, enter the password, and you're done! The device will show its IP address on screen.

🔧 Next Steps

1. Install the browser extension
2. Configure ESP32 IP or Telegram in extension settings
3. Visit your MakerWorld profile and enjoy live monitoring!

✨ Optional: Buzzer Setup

  • Connect passive buzzer positive (+) to GPIO 22
  • Connect buzzer negative (-) to GND
  • Enjoy sound notifications for every event!
  • No buzzer? No problem - everything works without it
